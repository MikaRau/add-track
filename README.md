# gsxr-track-add

Public HTTPS companion page for the GSX-R dash 'add track' feature. Open on an Android phone in Chrome, draw the start/finish line on the satellite map, and 'Send to dash' writes the two endpoints over Web Bluetooth (BLE) to the dash's GATT service. The dash app and BLE receiver (aa_bt_all.py) live in the private GSX-R repos.
